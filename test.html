üìã Integration Examples

Here are different ways to integrate the flexible widget:

1. Modal Widget (Recommended for marketing pages)

<!-- Include widget files --> <link rel="stylesheet" href="widget.css"> <script src="gm-booking-widget-standalone.js"></script> <!-- Add a button to trigger the modal --> <button onclick="openBookingModal()">Book Your Venue</button> <script> function openBookingModal() { GMBookingModal({ venue: 'manor', theme: 'light', defaultVenueArea: 'upstairs' }); } </script>

2. Inline Widget with Flexible Sizing

<!-- Small widget --> <div data-gm-widget="booking" data-venue="manor" data-theme="light" class="size-small"></div> <!-- Large widget --> <div data-gm-widget="booking" data-venue="both" data-theme="light" class="size-large"></div> <!-- Full width widget --> <div data-gm-widget="booking" data-venue="both" data-theme="light" class="size-full"></div>

3. Dark Theme Modal

<script> function openDarkModal() { GMBookingModal({ venue: 'both', theme: 'dark', defaultVenueArea: 'full_venue', showSpecialRequests: true }); } </script>

üîß Configuration Options

The widget now supports these flexible options:

  • Modal Mode: Use GMBookingModal() for overlay functionality
  • Sizing: size-small, size-large, size-full
  • Themes: light or dark
  • Venues: manor, hippie, or both
  • Areas: upstairs, downstairs, full_venue
